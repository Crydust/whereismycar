An HTML 5 "where is my car" application that stores your current location and helps you get back to a previously stored location.

1. install nodejs

2. install grunt-cli
$ sudo npm install grunt-cli -g

3. download development dependencies
$ npm install

4. download runtime dependencies
$ volo install

5. develop with typescript, sourcemaps, unittests with coverage
$ grunt dev
and open your browser at 
http://localhost:6001/src/
http://localhost:6001/test/

6. publish: test, add semicolons, lint, compile, concatenate, minify, make cachable ...
$ grunt publishAlmond
